# Auth_Routes
Authentication and Authorization using JWT tokens (NodeJs and Typescript)

GET:`/auth/allUsers`

Routes: <br>
GET:`/user/all`

Auth Routes<br>
POST:`/auth/register`<br>
POST:`/auth/login`<br>
POST:`/auth/logout`<br>
POST:`/auth/changepassword` (Authorized Using JWT tokens)<br>
